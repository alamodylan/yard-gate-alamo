Importar Chasis (Excel)

Plantilla esperada:
chassis_number | plate | length_ft | axles | type_code | status | chassis_kind | predio
chassis_number → número del chasis (5 dígitos). Ej: 40123
plate → placa del chasis (opcional).
length_ft → largo del chasis en pies (20 / 40 / 45) (opcional).
axles → cantidad de ejes (2 o 3) (opcional).
type_code → código interno (20FT_2AX / 20FT_3AX / 40FT_2AX / 40FT_3AX) (opcional).
status → estado del chasis: BUENO / DAÑADO / FUERA_DE_SERVICIO / ATADO (opcional, por defecto BUENO).
chassis_kind → tipo de chasis: CHASIS / LOW_BOY / TANQUETA / PLANA / CARRETA (opcional, por defecto CHASIS).
predio → nombre del predio (debe coincidir con el nombre del predio en el sistema). Si va vacío, se usa el predio activo.
Si length_ft y axles están vacíos, el sistema los deduce automáticamente según el prefijo del chasis.
Nota: si el predio escrito no existe exactamente, esa fila se marca como error y no se importa.
